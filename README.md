## Spring boot shoping list
### В процессе доработки
Приложение реализующее создание, изменение и удаления списка покупок, с упором на Backend. Использует технологии:
<hr>
<ul>
  <li>Java 17, Maven</li>
  <li>Spring boot 3</li>
  <li>Hibernate</li>
  <li>PostgreSQL, H2 database</li>
  <li>Lombok</li>
</ul>
